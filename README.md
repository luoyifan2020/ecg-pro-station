# ECG PRO 临床学习站

一个基于数据驱动的结构化心电图（ECG）学习工具，旨在通过典型的临床波形与中外医学标准对比，帮助初学者快速掌握心电图判读。

## 核心功能
- 📖 **结构化学习卡**：涵盖 30 种核心心电诊断，提供中外教学要点对比。
- 🔍 **实时检索**：支持中英文名、SCP 代码快速查询。
- 🧪 **自测模式**：随机抽题，建立你的错题本。
- 📁 **本地病例库**：支持上传个人临床病例，进行归档管理。

## 目录结构
- `/images`: 存放 30 张核心心电图示例。
- `data.json`: 存储所有医学描述数据的核心数据库。
- `index.html`: 前端展示页面。
- `/scripts`: 图片处理与数据同步工具。

## 鸣谢
数据来源参考 LITFL, ECGwaves 以及 SCP-ECG 标准标准。
